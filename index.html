CSS box model generally explains all the properties of a rectangular box. Each displayable component on a web page is composed of one or more rectangular boxes. CSS box models generally explain how these rectangular boxes are arranged up on a web page. These boxes can have varied features and interact with one another in different ways, but they all include a content area and optional surrounding padding, border, and margin regions.
